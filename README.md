# CiMFlowSim

Cycle-level SystemC/TLM dataflow simulator for weight-stationary compute-in-memory (CiM) accelerators.

Preliminary conference version: *eF2lowSim*, DATE 2023 (doi: 10.23919/DATE56975.2023.10137200).

## Requirements

Linux x86_64, SystemC 2.3.3, g++ (C++17), Python >= 3.9.
